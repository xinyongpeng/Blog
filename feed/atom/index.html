<p>引用p牛的一句话</p> <blockquote><p>一段代码，改变其中的变量，将会导致这段代码产⽣生功能性的变化，我称之为动态特性</p> </blockquote> <p>PHP本身有很多的动态特性，java虽然没有那么灵活，但是java的反射赋予了Java一些动态地特性</p> <p>我在学习java的时候，一直不太清楚java是如何获取变量的类型的，python和JavaScript都可以用<code>type</code>来分清楚这个变量的类型</p> <p>后来才知道<code>getClass</code>这个方法，可以返回变量的类型</p> <p>比如</p> <pre><code>System.out.println("a".getClass()); 输出class java.lang.String System.out.println(new Integer(1).getClass()); 输出class java.lang.Integer System.out.println(Runtime.getRuntime().getClass()); 输出 class java.lang.Runtime</code></pre> <p>这样就知道了这个对象属于哪个类</p> <p>其实我们不止可以获得对象属于哪个类，我们还可以调用这个对象所属的类的方法</p> <p>来看一个最常用的payload</p> <div class="highlight"><pre><span></span><span class="n">Class</span> <span class="n">clazz</span> <span <li>首先获取<code>java.lang.Runtime</code>的类型<code>类</code></li> <li>然后获取<code>java.lang.Runtime</code>的方法<code>exec</code>，由于<code>exec</code>方法有重载，所以要指明参数类型<code>String.class</code></li> <li>之后通过<code>invoke</code>调用，第一个参数是<code>Runtime</code>的对象，第二个参数是<code>calc</code></li> </ol> <p>invoke调用时候的规定如下：</p> <ol> <li>如果是一个普通方法，第一个参数是类对象</li> <li>如果是一个静态方法，第一个参数是类</li> </ol> <p><code>java.lang.Runtime</code>的构造函数是私有的，也就是说不能通过实例化来产生其对象，必须通过调用<code>java.lang.Runtime.getRuntime</code>方法来得到一个对象</p> <p>同时<code>getRuntime</code>方法是静态方法，所以<code>invoke</code>的第一个参数是类对象，没有其他的参数了</p> <p>单例模式是因为有些操作只需要做一次就够了，比如建立数据库连接之类的</p> <p>故其实中还有一次反射调用</p> <ol> <li>获取<code>java.lang.Runtime</code>的方法<code>getRuntime</code></li> <li>通过invoke调用，产生一个<code>Runtime</code>的实例对象</li> </ol> <h2>反序列化</h2>
